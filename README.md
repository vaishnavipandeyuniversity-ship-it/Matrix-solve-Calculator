# Matrix-solve-Calculator
A simple Matrix Calculator built using the C programming language.
This program performs multiple matrix operations such as:
Matrix Multiplication
Matrix Addition
Matrix Subtraction
Matrix Transpose
Scalar Multiplication

Designed for beginners learning C programming, arrays, and matrix operations.

Features
✔ Input two matrices
✔ Display matrices
✔ Matrix Multiplication
✔ Matrix Addition
✔ Matrix Subtraction
✔ Matrix Transpose
✔ Scalar Multiplication
✔ Beginner-friendly code structure

Operations Available
1. Matrix Multiplication
Condition: Am×n​×Bn×p​
	​
2. Matrix Addition
Condition:
Both matrices must have same rows and columns.
A+B

3. Matrix Subtraction
Condition:
Both matrices must have same dimensions.
A−B

4. Matrix Transpose
Transpose changes rows into columns.
AT

5. Scalar Multiplication
Each element of matrix is multiplied by a scalar number.
kA
